Extension: ExtensionUKCoreMainLocation

An extension to support a reference to the main location for an organisation.

The official URL for this extension is: 

https://fhir.nhs.uk/R4/StructureDefinition/Extension-UKCore-MainLocation

Status: draft
Extension maintained by: Project Wildfyre

Context of Use

This extension may be used on the following element(s):

  • {"type"=>"Organization"}

Usage info

Usage:

  • This Extension is not used in this Implementation Guide
